Nombre del proyecto: TacoExpress App

Descripción breve de la app:
Aplicación móvil desarrollada para facilitar la realización de pedidos en una taquería de forma rápida, sencilla y organizada.
La app permitirá a los clientes consultar el menú digital, agregar productos al carrito y realizar pedidos desde su celular. Además, ayudará al negocio a mejorar la organización de pedidos y reducir errores.

Tecnología / framework elegido:
- Flutter
- Dart
- Firebase
- GitHub
